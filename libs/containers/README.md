# Crypto3.Containers

Merkle trees and related cryptographic containers for the Crypto3 suite.

This header-only component is maintained as part of the Crypto3 monorepo. See
the [root build instructions](../../README.md#clone-and-build) to configure the
project and run its tests. Link the component as `crypto3::containers` from
CMake.

Buildable examples are available under [`example`](example/).
