# Crypto3.Hash

Hash function implementations for the Crypto3 suite.

This header-only component is maintained as part of the Crypto3 monorepo. See
the [root build instructions](../../README.md#clone-and-build) to configure the
project and run its tests. Link the component as `crypto3::hash` from CMake.

API notes and examples are available under [`docs`](docs/), and buildable
examples are under [`example`](example/).
