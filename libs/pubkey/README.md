# Crypto3.Pubkey

Public-key signature, encryption, and secret-sharing schemes for the Crypto3
suite.

This header-only component is maintained as part of the Crypto3 monorepo. See
the [root build instructions](../../README.md#clone-and-build) to configure the
project and run its tests. Link the component as `crypto3::pubkey` from CMake.

API notes are available under [`docs`](docs/), and the maintained BLS example is
under [`example`](example/).
